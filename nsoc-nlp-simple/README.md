# nlp
一.简述
-----------------------------------------
本项目主要实现了一段文本的关键字提取，反馈功能。

1.本项目使用了word分词器;

2.关键字提取部分采用了TF-IDF算法思想;

3.关键字提取出来以后，可以有一个feedback方法，将想要强化的关键字强化，待下一次出现时会有更高的评分；

二.如何使用
----------------------------------------
本项目分为客户端和服务端,服务端打包后可直接java -jar ... 运行，调用方面，通过引用客户端jar包，具体使用可参照NClientTest.java。
